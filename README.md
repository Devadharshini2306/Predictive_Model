# Predictive Analysis using Machine Learning

## Project Overview

This project demonstrates predictive analysis using Machine Learning.
The goal is to build a model that can predict house prices based on various input features.

## Objectives

* Perform data preprocessing and cleaning
* Select relevant features for prediction
* Build a Machine Learning model
* Evaluate model performance
* Visualize prediction results

## Tools & Technologies

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Google Colab

## Dataset

The dataset used is a house price dataset (`train.csv`) which includes:

* Numerical features (area, rooms, quality, etc.)
* Target variable: `SalePrice`

## Steps Performed

1. Loaded the dataset using Pandas
2. Explored the dataset using `.info()` and `.describe()`
3. Selected only numerical features
4. Handled missing values using mean filling
5. Split data into training and testing sets
6. Trained a Linear Regression model
7. Predicted house prices
8. Evaluated the model using Mean Squared Error
9. Visualized actual vs predicted values

## Model Used

* Linear Regression

## Key Insights

* House prices depend on multiple features like area and quality
* The model can predict prices with reasonable accuracy
* Data preprocessing improves model performance

## Output

* Predicted house prices
* Model evaluation metric (MSE)
* Visualization of actual vs predicted values
